About

Managing information and tasks is still a difficult problem for most. I, Sushrut Bidwai personally feel, no single tool available does this elegantly. I built Simplycious to solve this problem at least for myself.

Inspiration behind Simplycious is Twitter, in the way it is a simple and yet powerful tool to consume and propagate news/views.

How it works?

You create a textual message and tag it with any number of appropriate tags. The message could be a bookmark, task, todo, link - almost any thing.

Tags (any word starting with a #) is way you categories these messages. For example, I have created tags like books, movies, todo, buy, expenses.

Special or System tags

There are few built in tags which have special significance and functionality attached to them. If you want to use the special functionality, then it has to be a first tag in the message.

  • #expenses: Helps in keeping track of expenses either personal or professional. There are two states for it - reported and reimbursed.
  • #buy: Always forget what you want to buy when you visit a grocery store? Then use this to keep a note of shopping list.
  • #bookmark: There is no special functionality yet, but I few plans regarding this.
  • #readlater: Helps in managing current reading list.
  • #todo: Simple task management. Done with a task, mark it done. Collaboration features for this in the pipeline.

#wiki system tag

#wiki is a system tag which helps you in categories information very easily. You can use this tag along with any number of other tags.

As you keep adding information to wiki, over a period of time, it starts understanding depth and breadth of the information. Alows you for auto-nesting of categories.

Some examples of where this will be helpful.

  • If you are an entrepreneur and wants to keep track of all ideas, use it like #wiki #ideas #more $tags.
  • If you found new unix commands you want to remember for a rainy day, #wiki #unix #commands ...
  • Organize quotes that you like.
  • So on. Pretty much any thing you would like to keep record of for easy look up in future.

Tricks

  • Create tags like #daily , #mondays etc for tasks which repeat.
  • I use #wiki tag along with few other things to keep track important information to me and my business. Eg #wiki #competition tadalist.com
