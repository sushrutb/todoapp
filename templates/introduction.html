About

Managing information and tasks is difficult problem. I, Sushrut Bidwai, feel, no single tool available does this elegantly. I built Simplycious to solve this problem at least for myself.

We consume lot of information and content. Certain part of it is useful which we would like to record and use it afterwards. It can be as simple as bookmarking some article, quote from a book, like to a recipe or news about your competitor.

Simplycious is built in a way that, it lets you record/save any type of information you want and then easily find that again. I use to keep track of my daily tasks (except meetings), bookmarks, reading list, movies I want to see. I keep track of new and interesting products/startups using it.
